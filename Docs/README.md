# Varneth – Text Adventure RPG Engine

Ein modulares, zustandsbasiertes Text-Adventure / RPG in Java.
Fokus: saubere Architektur, Erweiterbarkeit, Trennung von Definition und State.

## Features
- Actor / Enemy / Player System (Definition + State)
- Combat- und Exploration-Phasen
- Dialog-Chunk-System
- Skill- & Spell-System
- Welt- und Raumlogik

## Ziel des Projekts
- Lernprojekt für OOP, Java, Architektur
- Später: UI, Lokalisierung, mobile-fokus

## Tech-Stack
- Java
- IntelliJ IDEA
